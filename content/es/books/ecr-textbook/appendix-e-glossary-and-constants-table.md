### **E.2 Ecuaciones Canónicas**

1. **Reciprocidad Entropía-Coherencia:** $dS + k_* d\ln C = 0$
2. **Bomba de Coherencia (Forma Dinámica):** $\dot C = \alpha (F - S_C)$
3. **Fórmula de Resonancia Adaptativa (ARF):** $\Delta S = R \Psi C$
4. **Invariante Global:** $\sum_{\mu} (dS_{\mu} + k_*^{(\mu)} d\ln C_{\mu}) = 0$
5. **Energía de Coherencia:** $E_{coh} = T k_* \ln C$

| Término | Definición Concisa |
| :--- | :--- |
| **Homeoresonancia** | Estado estable donde la salida de entropía equilibra la entrada de coherencia (α ≈ 0). |
| **Bomba de Coherencia** | Cualquier sistema abierto que convierte energía entrante en estructura organizada. |
| **Bucle de Resonancia** | Ciclo de retroalimentación entre niveles-μ que produce coherencia autosostenida. |
| **Bloqueo de Fase** | Sincronización de osciladores a través de niveles. |

| **Campo Noético** | Dominio de resonancia μ₆–μ₇ a escala planetaria. |
