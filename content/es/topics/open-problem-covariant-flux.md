# Cerrando el círculo con la gravedad

FRC 893 propone que "la gravedad es el influjo del potencial de vacío". FRC 566 define el "Flujo de Coherencia".

**El problema abierto:**
Fusiónalos. Necesitamos las ecuaciones de campo de Einstein-Coherencia:
$$ G_{} = 8 G (T_{}^{materia} + T_{}^{}) 
$$ donde $T_{}^{}$ se deriva directamente del flujo covariante $J^_C$.

Resolver este problema demostraría matemáticamente que la "Gravedad Metabólica" recupera las ecuaciones de campo de Einstein en el límite macroscópico.
