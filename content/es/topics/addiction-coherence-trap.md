# Adicción: Una trampa de coherencia

La adicción es una trampa de coherencia. La aplicación más precisa de dS + k·d ln C = 0 a escala personal.

## Paso 1: El primer impacto — Alfa artificial

Una sustancia (o comportamiento — juego, porno, redes sociales, comida) crea un pico masivo de alfa artificial. El aumento de dopamina es un error de predicción de recompensa. En términos de CGL: la puerta del aprendizaje se abre de par en par. El sistema se reorganiza alrededor de esta nueva señal. Con fuerza. Rápido. Profundo.

Esto es formación de coherencia. Construcción real de patrones. Las vías señal->droga->recompensa se cristalizan.

## Paso 2: La puerta se estrecha

El sistema se adapta. Se genera tolerancia. La misma dosis produce menos alfa. La puerta empieza a cerrarse para todo excepto para la sustancia. Las recompensas naturales se vuelven grises. El perfil de sensibilidad alfa se estrecha a una sola frecuencia. La puerta solo se abre para una llave.

## Paso 3: La abstinencia como coherencia negativa

Sin la sustancia, el sistema entra en alfa negativo: ansiedad, disforia, dolor. En términos de FRC: la coherencia cae rápido, la entropía aumenta rápido. Por eso la abstinencia se siente como morir.

## Paso 4: Compulsión — Persiguiendo el cero

El adicto ya no persigue el "subidón". Persigue el estado base. La sustancia previene el alfa negativo. El sistema ha externalizado el mantenimiento de la coherencia a una entrada externa.

## Recuperación (Visión FRC)

La recuperación no es solo detenerse. Eso desencadena la cascada de entropía. La recuperación consiste en reabrir la puerta a múltiples fuentes de coherencia: comunidad, naturaleza, ejercicio, meditación, conexión. No fuerza de voluntad. Nueva estructura más fuerte que el atractor de la adicción.
