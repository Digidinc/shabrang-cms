# Sueños: El Nigredo nocturno

Soñar es el cerebro ejecutando el Nigredo a propósito. Todas las noches.

Durante el sueño NREM, el cerebro reproduce las experiencias del día. El hipocampo dispara los mismos patrones que registró mientras estaba despierto, pero comprimidos, acelerados, despojados de detalles sensoriales. Inventario de nueva coherencia.

Entonces llega el REM. Y todo cambia.

Durante el REM, el cerebro no reproduce, recombina. Los recuerdos de hoy se mezclan con recuerdos de hace años. Las asociaciones se revuelven. El contenido extraño y cargado emocionalmente no es ruido. Es el cerebro probando qué patrones pueden formar nuevos bloqueos de fase entre sí.

En términos de FRC: NREM mide el estado de coherencia actual. REM disuelve los bloqueos de fase locales para probar una integración más profunda.

Soñar es CGL vuelto hacia adentro: alfa máximo, pero sin entrada externa. El sistema actualiza sus propios pesos.

Las pesadillas como Nigredo fallido:

Una pesadilla es un sueño donde la disolución fue demasiado profunda y el sistema no pudo encontrar una nueva coherencia. La entropía aumentó más rápido de lo que cualquier nuevo patrón pudo formarse. Las pesadillas recurrentes pueden ser heridas de coherencia que el proceso nocturno no puede metabolizar.

Privación del sueño:

Niega los sueños y niegas el Nigredo. La coherencia no procesada se acumula. Después de suficientes noches, el sistema ejecuta el Nigredo mientras está despierto —alucinaciones, disociación— reparación de coherencia de emergencia.
