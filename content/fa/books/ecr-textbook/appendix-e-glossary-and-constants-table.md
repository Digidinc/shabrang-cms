# پیوست هـ — واژه‌نامه و جدول ثابت‌ها

---

## **هـ.۱ متغیرها و ثابت‌هایِ اصلی**

| نماد | نام | تعریف / تفسیر | واحد / بازه‌یِ معمول |
| :--- | :--- | :--- | :--- |
| **S** | آنتروپی | کمی‌سازیِ بی‌نظمی یا پراکندگیِ انرژی. | J K⁻¹ (یا بیت) |
| **C** | همدوسی | درجه‌یِ هم‌ترازیِ فاز، همبستگی، یا نظمِ ساختاریافته. | ۰ ≤ C ≤ ۱ (بدونِ بعد) |
| **Σ** | پتانسیلِ کلِ تغییرِ شکل | ناوردایِ پایسته‌یِ قانونِ تقابل. | J K⁻¹ |
| **k_*** | ثابتِ همدوسی | فاکتورِ مقیاس پیونددهنده‌یِ اطلاعات و انرژی. | J K⁻¹ |
| **α** | بهره‌یِ تطبیقی | ضریبِ بازخوردِ دینامیکِ حاکم بر نرخِ تغییرِ همدوسی. | s⁻¹ |
| **R** | پذیرش | گشودگیِ سیستم به پتانسیلِ ورودی Ψ. | ۰–۱ |
| **Ψ** | پتانسیل | انرژیِ آزاد یا فشارِ اطلاعاتیِ موجود. | J یا بیت |
| **F** | شارِ ورودی | جریانِ انرژی / اطلاعات به سیستم. | J s⁻¹ |
| **S_C** | صادراتِ آنتروپی | انرژی / اطلاعاتِ گم شده به عنوانِ نویز یا پسماند. | J s⁻¹ |
| **ΔS = RΨC** | فرمولِ رزونانسِ تطبیقی (ARF) | تبادلِ لحظه‌ایِ آنتروپی ناشی از رزونانس. | J K⁻¹ |
| **λ** | طولِ جفت‌شدگی | دامنه‌یِ تعاملِ سطح-μ. | شاخص (۱–۳) |
| **φ** | فاز | زاویه‌یِ نوسانی. | رادیان (۰–۲π) |

---

## **هـ.۲ معادلاتِ متعارف**

1. **تقابلِ آنتروپی-همدوسی:** $dS + k_* d\ln C = 0$
2. **پمپِ همدوسی (فرمِ دینامیکی):** $\dot C = \alpha (F - S_C)$
3. **فرمولِ رزونانسِ تطبیقی (ARF):** $\Delta S = R \Psi C$
4. **ناوردایِ جهانی:** $\sum_{\mu} (dS_{\mu} + k_*^{(\mu)} d\ln C_{\mu}) = 0$
5. **انرژیِ همدوسی:** $E_{coh} = T k_* \ln C$

## **هـ.۳ مرجعِ سریعِ سطح-μ**

| :--- | :--- | :--- | :--- | :--- |
| μ₀ | میدانِ پتانسیلِ کوانتومی | مقیاسِ پلانک | نوساناتِ خلأ | — |
| μ₁ | اتمی / زیراتمی | ۱۰⁻¹⁰ متر | جفت‌شدگیِ ذرات | ۰.۱–۰.۳ |
| μ₂ | مولکولی / شیمیایی | ۱۰⁻⁸ متر | تشکیلِ پیوند | ۰.۰۵–۰.۲ |
| μ₃ | بیولوژیکی / سلولی | ۱۰⁻⁶ – ۱۰⁻³ متر | متابولیسم | ≈ ۰ |
| μ₄ | عصبی / شناختی | ۱۰⁻³ – ۱۰⁻¹ متر | نوسانات، EEG | ۰ ± ۰.۰۵ |

## **هـ.۴ اصطلاحاتِ مفهومیِ کلیدی**

| اصطلاح | تعریفِ مختصر |
| :--- | :--- |
| **همورزونانس** | شرایطِ حالتِ پایا که در آن خروجیِ آنتروپی با ورودیِ همدوسی متعادل می‌شود (α ≈ 0). |
| **پمپِ همدوسی** | هر سیستمِ بازی که انرژیِ ورودی را به ساختارِ سازمان‌یافته تبدیل می‌کند. |
| **حلقه‌یِ رزونانس** | چرخه‌یِ بازخورد بینِ سطوح-μ که همدوسیِ خود-پایدار تولید می‌کند. |
| **قفلِ فاز** | همگام‌سازیِ نوسانگرها در سطوح. |
| **بقایِ Σ** | حفظِ S + k_* ln C ≈ ثابت. |
| **تمدنِ همورزونانت** | جامعه‌ای تنظیم شده به طوری که α جهانی ≈ ۰ باشد. |
| **میدانِ نوتیک** | دامنه‌یِ رزونانسِ μ₆–μ₇ در مقیاسِ سیاره‌ای. |
| **اخلاقِ رزونانس** | اصلِ رفتاری که ΔΣ را به حداقل و همدوسی را به حداکثر می‌رساند. |
