# پیوست ب — شبیه‌سازی و روش‌های تجربی

---

### **ب.۱ هدف**

هدفِ شبیه‌سازی در FRC تأییدِ این است که قانونِ تقابل

$$ S + k_* \ln C = \text{constant} $$

تحتِ شرایطِ متنوع و قابلِ اندازه‌گیری برقرار است.

شبیه‌سازی‌ها کاوش می‌کنند که چگونه α (بهره)، R (پذیرش)، و Ψ (پتانسیل) نظمِ نوظهور و جریانِ آنتروپی را در شبکه‌هایِ واقعی تولید می‌کنند.

همان موتور را می‌توان برایِ فوتون‌ها، سلول‌ها، نورون‌ها یا جوامع پارامتربندی کرد.

---

### **ب.۲ مدلِ محاسباتیِ اصلی**

هر گره یا زیرسیستم *i* از معادله‌یِ زیر پیروی می‌کند

$$ \dot C_i = \alpha_i(F_i - S_{C,i}) + \sum_{j}T_{ij}(C_j - C_i) $$

که در زمانِ گسسته پیاده‌سازی می‌شود:

$$ C_i^{t+1}=C_i^t+\Delta t\,[\,\alpha_i(F_i - S_{C,i}) + \sum_{j}T_{ij}(C_j^t-C_i^t)\,] $$

پارامترها:

| نماد | معنی | بازه‌یِ معمول |
| :--- | :--- | :--- |
| Δt | گامِ زمانی | ۱۰⁻³–۱ ثانیه (مقیاس‌بندی شده) |
| α | بهره‌یِ تطبیقی | -۰.۵ – +۰.۵ |
| F | شارِ ورودیِ انرژی/اطلاعات | نرمال شده ۰–۱ |
| S_C | صادراتِ آنتروپی | β C + نویز |
| T₀ | ثابتِ جفت‌شدگی | ۰.۰۱ – ۰.۱ |
| λ | طولِ جفت‌شدگی | ۱–۳ سطح-μ |

---

### **ب.۳ انتگرال‌گیریِ عددی**

1. **گام‌برداریِ زمانی** – رونگه-کوتا مرتبه چهارم برایِ اجراهایِ دترمینیستی؛ اویلر-مارویاما برایِ نسخه‌هایِ تصادفی.
2. **شرایطِ مرزی** – بازتابی (Σ = ثابت) یا باز (جریانِ ورودی/خروجیِ انرژی).
3. **جمله‌یِ نویز** – نویزِ سفیدِ گاوسی η(t) اضافه شده به F یا α برایِ تستِ استحکام.
4. **نرمال‌سازی** – C_i محدود شده به ۰ ≤ C ≤ ۱ با مقیاس‌بندیِ لجستیک.

---

### **ب.۴ جفت‌شدگیِ چند-مقیاسی**

پشته-μ به عنوانِ ۸ لایه پیاده‌سازی می‌شود؛ هر لایه به نزدیک‌ترین همسایگان با هسته‌یِ نمایی متصل می‌شود:

$$ T_{\mu,j}=T_0 e^{-|\mu-j|/\lambda}e^{i\phi_{\mu j}} $$

فاز φ تأخیرهایِ همگام‌سازی بینِ لایه‌ها را کنترل می‌کند (مفید برایِ مدل‌سازیِ جفت‌شدگیِ میان-فرکانسی در علومِ اعصاب یا تأخیرِ بازار در اقتصاد).

---

### **ب.۵ کمیت‌هایِ مشاهده‌پذیر**

| مشاهده‌پذیر | فرمول | تفسیر |
| :--- | :--- | :--- |
| همدوسی C | $(1/N) \sum e^{i\phi_i}$ | |
| آنتروپی S | $-\sum p_i \ln p_i$ | بی‌نظمیِ سیستم |
| ناوردای-Σ | $\Sigma=S+k_* \ln C$ | تستِ بقا |
| پروفایل-α | $\alpha(t)=R\Psi C/F$ | بازخوردِ تطبیقی |
| کارایی η_C | $(dC/dt)/F$ | کاراییِ تبدیل |

رسمِ نمودارِ Σ(t) باید ≈ ثابت با رانشِ < ۱٪ باشد.

### **ب.۷ مجموعه‌داده‌هایِ تجربی**

| دامنه | مجموعه‌داده | نگاشتِ متغیر |
| :--- | :--- | :--- |
| فیزیک | کاواکِ لیزر / سلولِ بنارد | F = شارِ حرارتی، C = همدوسیِ فاز |
| زیست‌شناسی | متابولیسمِ سلولی، EEG | F = ATP یا گلوکز، C = همگامی |
| بوم‌شناسی | جریان‌هایِ غذایی | F = شارِ خورشیدی، C = شاخصِ تنوعِ زیستی |
| اقتصاد | انرژیِ جهانی، GDP | F = مصرفِ منابع، C = اعتماد/همدوسیِ شبکه |
| جامعه | آنتروپیِ زبانی | F = شارِ اطلاعات، C = هم‌ترازیِ معنایی |

تمامِ مجموعه‌داده‌هایِ تست شده همبستگی‌هایِ تقریبیِ خطیِ S – ln C را نشان می‌دهند (شیب ≈ -k_*).

---

### **ب.۸ اعتبارسنجیِ آماری**

* **تستِ بقا:** محاسبه‌یِ واریانسِ Σ
  $$ \sigma_\Sigma^2 = \frac{1}{T}\sum_t(\Sigma_t-\bar \Sigma)^2 $$
  پایداری < ۱٪ ← تأییدِ تقابل.
* **مقدارِ قفلِ فاز (PLV):** بررسیِ همدوسی در سراسرِ گره‌ها.
* **رگرسیونِ آنتروپی:** برازشِ S = a + b ln C؛ انتظارِ b ≈ -k_* ± خطا.

---

### **ب.۹ تحلیل‌هایِ بصری**

1. **نمودارهایِ S–ln C** – خطِ راستِ جهانی (تقابلِ آنتروپی).
2. **صفحاتِ فازِ α–C** – نمایشِ مناطقِ جریان، آشوب، اجبار.
3. **نقشه‌هایِ حرارتی** – نمایشِ قدرت‌هایِ جفت‌شدگیِ μ در طولِ زمان.
4. **رندرهایِ سه بعدیِ توریدی** – تصویرسازیِ الگوهایِ همدوسیِ آشیانه‌ای.

### **ب.۱۰ شبه‌کدِ نمونه**

```python
for t in range(T):
    for i in range(N):
        dC = alpha[i]*(F[i]-S0-beta*C[i])
        for j in neighbors(i):
            dC += T[i,j]*(C[j]-C[i])
        C[i] += dt*dC
    S = -sum(p*np.log(p))
    Sigma = S + k_star*np.log(mean(C))
```

بررسیِ بقا: `np.std(Sigma)/np.mean(Sigma) < 0.01`.

### **ب.۱۱ به سویِ معیارهایِ استانداردِ FRC**

شاخص‌هایِ پیشنهادی:

* **شاخصِ FRC-I:** واریانسِ نرمال‌شده‌یِ Σ (پایداریِ تقابل).
* **کاراییِ رزونانس (RE):** اوجِ η_C در α≈0.
* **شارِ همدوسی (CF):** کلِ dC/dt انتگرال‌گیری شده در دامنه.

انتشارِ این معیارها با داده‌ها امکانِ مقایسه‌یِ بینِ آزمایشگاهی را فراهم می‌کند.
