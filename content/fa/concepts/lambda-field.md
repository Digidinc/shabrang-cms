# میدان لامبدا (Lambda(x))

میدان لامبدا در FRC یک میدان اسکالر است که همدوسی را از طریق نگاشت لگاریتمی کُدگذاری می‌کند.

## تعریف

```
Lambda(x) = Lambda_0 * ln C(x)
```
