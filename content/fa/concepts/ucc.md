# شرط همدوسی جهانی (UCC)

شرط همدوسی جهانی (UCC) قانون جریان محلی است که در FRC برای توصیف چگونگی تغییر همدوسی در فضا و زمان استفاده می‌شود.

## شکل متعارف جریان

```
d/dt ln C = -div(J_C) + S_C
J_C = -D_C grad(ln C)
```

تفسیر:
- `J_C` شار همدوسی (انتقال) است.
- `S_C` منابع/چاه‌ها (تولید یا اتلاف) هستند.
- `D_C > 0` قدرت نفوذ در فضای لگاریتم-همدوسی را تعیین می‌کند.
