# شرط همدوسی جهانی (UCC)

در چارچوب FRC، شرط همدوسی جهانی (UCC) یک قانون جریان محلی است که توضیح می‌دهد
همدوسی چگونه در فضا و زمان تغییر می‌کند.

## فرم استاندارد

```
d/dt ln C = -div(J_C) + S_C
J_C = -D_C grad(ln C)
```
