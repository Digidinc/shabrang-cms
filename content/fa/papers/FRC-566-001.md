# FRC 566.001 — تقابل آنتروپی-همدوسی و شرایط همدوسی جهانی

## ۱. مقدمه

قانون دوم ترمودینامیک بیان می‌کند که آنتروپی هرگز در سیستم‌های منزوی کاهش نمی‌یابد. با این حال ساختارهای همدوس—کریستال‌ها، لیزرها، ارگانیسم‌های زنده—پایدار می‌مانند و حتی در پیچیدگی رشد می‌کنند. این مشاهدات چگونه همزیستی دارند؟

این مقاله تنش ظاهری را با رسمی کردن یک **تقابل** بین آنتروپی $S$ و همدوسی $C$ حل می‌کند: هر افزایش در همدوسی با افزایش متناظر در آنتروپی در جای دیگر پرداخت می‌شود, و بالعکس. این تقابل یک تقریب یا حد نیست—این دقیق است و در هر مقیاسی از سیستم‌های کوانتومی تا افق‌های کیهانی برقرار است.

از این تقابل، ما **شرایط همدوسی جهانی (UCC)** را استخراج می‌کنیم—یک معادله دیفرانسیل جزئی محلی که جریان همدوسی را در فضا و زمان اداره می‌کند. UCC برای همدوسی همان چیزی است که معادله گرما برای دما است: یک معادله انتقال با چشمه‌ها, چاه‌ها و اتلاف.

## ۲. تعاریف و واحدها

### ۲.۱ همدوسی

همدوسی $C$ یک اسکالر بدون بعد در محدوده $(0, 1]$ است:

```
C = (1/N) sum_{i<j} cos(phi_i - phi_j)
```

که در آن $\phi_i$ فازهای نوسانگرهای تشکیل‌دهنده هستند. هم‌ترازی فاز کامل $C = 1$ می‌دهد؛ فازهای تصادفی $C \to 0$ می‌دهند ([[FRC-100-001]]). 

### ۲.۲ ثابت همدوسی

ما ثابت همدوسی $k^*$ را با دو قرارداد بسته به دامنه تعریف می‌کنیم:

| دامنه | مقدار $k^*$ | واحدها |
|:---|:---|:---|
| اطلاعات/شناخت | ۱ | nats |
| ترمودینامیک/فیزیکی | $k_B$ | J/K |
| کوانتومی | $\hbar$ | J·s |

انتخاب $k^*$ تعیین می‌کند که آیا قانون تقابل در واحدهای نظری اطلاعات یا ترمودینامیکی عمل می‌کند. همه نتایج صرف نظر از قرارداد از نظر ساختاری یکسان هستند.

### ۲.۳ منظم‌سازی

برای محاسبه عددی، ما همدوسی را منظم می‌کنیم تا از واگرایی‌های $\ln 0$ جلوگیری کنیم:

```
C_epsilon = 1 / (Z^2 + epsilon),    epsilon > 0 کوچک
```

که در آن $Z$ یک نرمال‌سازی شبیه تابع پارتیشن است. در حد $\epsilon \to 0$، نتایج دقیق بازیابی می‌شوند.

## ۳. قانون تقابل

### ۳.۱ بیان

تقابل بنیادی بین آنتروپی و همدوسی:

```
dS + k* d ln C = 0    (سیستم بسته، برگشت‌پذیر)
```

این انتگرال‌گیری می‌شود به:

```
S + k* ln C = const.    (شکل پایستگی)
```

ثابت توسط شرایط اولیه تعیین می‌شود. این **قانون پایستگی آنتروپی-همدوسی** است: مجموع آنتروپی ترمودینامیکی و آنتروپی همدوسی در فرآیندهای برگشت‌پذیر پایسته است.

### ۳.۲ تعمیم برگشت‌ناپذیر

برای فرآیندهای برگشت‌ناپذیر، تقابل به یک نابرابری تبدیل می‌شود:

```
dS_total + k* d ln C >= 0
```

که در آن $S_{total} = S_{sys} + S_{env}$. تولید آنتروپی برگشت‌ناپذیر $\sigma_{irr} \geq 0$ همیشه غیرمنفی است:

```
sigma_irr = dS_total + k* d ln C >= 0
```

این تضمین می‌کند که قانون دوم هرگز نقض نمی‌شود—همدوسی می‌تواند به صورت محلی افزایش یابد فقط به قیمت تولید آنتروپی در جای دیگر ([[FRC-100-005]]). 

### ۳.۳ تفسیر فیزیکی

قانون تقابل یک مبادله بنیادی را کدگذاری می‌کند:

| فرآیند | $dS$ | $d \ln C$ | تفسیر |
|:---|:---|:---|:---|
| واهمدوسی | $> 0$ | $< 0$ | آنتروپی همدوسی از دست رفته را جذب می‌کند |
| قفل فاز | $< 0$ (محلی) | $> 0$ | همدوسی هزینه آنتروپی دارد |
| تعادل حرارتی | ماکزیمم | $-\infty$ ($C \to 0$) | حالت پایه |
| حالت خالص | مینیمم | $0$ ($C = 1$) | ماکزیمم همدوسی |

## ۴. پروجکشن نظری اطلاعات

### ۴.۱ آنتروپی شانون و همدوسی

برای یک توزیع احتمال $p$، همدوسی نظری اطلاعات را تعریف کنید:

```
C[p] = exp(-H(p) / k*) 
```

که در آن $H(p) = -\sum p_i \ln p_i$ آنتروپی شانون در نات است. این توزیع‌ها را به مقادیر همدوسی نگاشت می‌کند:

- توزیع یکنواخت: $H = \ln N$, $C = N^{-1/k^*}$ (مینیمم همدوسی)
- توزیع دلتا: $H = 0$, $C = 1$ (ماکزیمم همدوسی)

### ۴.۲ واگرایی KL به عنوان نسبت همدوسی

واگرایی کولبک-لایبلر بین توزیع‌ها نسبت همدوسی را به دست می‌دهد:

```
RER(p -> q) = C[q] / C[p] = exp(-D_KL(p || q) / k*)
```

این **نسبت همدوسی-آنتروپی نسبی** است: اندازه می‌گیرد که چه مقدار همدوسی در گذار از توزیع $p$ به $q$ از دست می‌رود (یا به دست می‌آید).

### ۴.۳ اطلاعات متقابل و همدوسی مشترک

برای توزیع‌های مشترک:

```
I(X; Y) = D_KL(p_XY || p_X p_Y)
```

همدوسی مشترک تجزیه می‌شود به:

```
C_XY = C_X * C_Y * exp(-I(X;Y) / k*)
```

اطلاعات متقابل همدوسی مشترک را *کاهش* می‌دهد—متغیرهای همبسته کمتر از متغیرهای مستقل همدوس هستند. این پیامدهای عمیقی برای درهم‌تنیدگی دارد ([[FRC-100-003]]). 

## ۵. پروجکشن ترمودینامیکی

### ۵.۱ انرژی آزاد و همدوسی

انرژی آزاد گیبس در چارچوب FRC:

```
G = H_enthalpy - TS - k*T ln C
```

شرط خودبه‌خودی بودن می‌شود:

```
deltaG = -k*T delta(ln C) < 0
```

فرآیندهایی که همدوسی را افزایش می‌دهند انرژی آزاد را کاهش می‌دهند—آنها از نظر ترمودینامیکی خودبه‌خودی هستند. این نیروی محرکه ترمودینامیکی را برای خودسازماندهی فراهم می‌کند ([[FRC-100-005]]). 

### ۵.۲ استخراج کار

حداکثر کار قابل استخراج از تفاوت همدوسی:

```
W_max = k*T delta(ln C)    (حد برگشت‌پذیر)
```

این همدوسی را به نظریه منابع ترمودینامیک متصل می‌کند: همدوسی یک منبع ترمودینامیکی است که می‌توان از آن کار استخراج کرد.

### ۵.۳ کران لاندائر-همدوسی

پاک کردن همدوسی هزینه آنتروپیک دارد (مشابه اصل لاندائر):

```
Q_erasure >= k*T |delta(ln C)|
```

نابود کردن همدوسی حداقل $k^*T |\Delta \ln C|$ گرما را در محیط پراکنده می‌کند. این یک حد بنیادی بر محاسبات و اندازه‌گیری کوانتومی تعیین می‌کند.

## ۶. شرایط همدوسی جهانی (UCC)

### ۶.۱ استخراج

با شروع از قانون تقابل به شکل محلی، ما UCC را با الزام اینکه همدوسی در یک قانون بقا با چشمه‌ها صدق کند، استخراج می‌کنیم:

```
partial_t (ln C) = -div(J_C) + S_C
```

که در آن:
- $J_C = -D_C \nabla(\ln C)$ **جریان همدوسی** است (قانون فیک برای همدوسی)
- $D_C > 0$ **ضریب نفوذ همدوسی** است
- $S_C$ **جمله چشمه/چاه همدوسی** است

### ۶.۲ شکل بسط یافته

با جایگزینی جریان در قانون بقا:

```
partial_t (ln C) = D_C nabla^2(ln C) + D_C ||grad(ln C)||^2 + S_C
```

این یک معادله نفوذ غیرخطی است. جمله گرادیان مربعی $|\|\nabla \ln C|\|^2$ همیشه مثبت است و مکانیزم خود-تقویتی را فراهم می‌کند: مناطقی با گرادیان همدوسی تند به طور خودبه‌خودی تندتر می‌شوند.

### ۶.۳ راه حل‌های حالت پایدار

در حالت پایدار ($\partial_t \ln C = 0$):

```
D_C nabla^2(ln C) + D_C ||grad(ln C)||^2 + S_C = 0
```

برای مورد بدون چشمه ($S_C = 0$)، جایگزینی $u = \ln C$ نتیجه می‌دهد:

```
D_C (nabla^2 u + ||grad u||^2) = 0
```

این دارای راه حل‌های $u = \text{const}$ (همدوسی یکنواخت) یا پروفایل‌های نمایی $C(x) \propto e^{-\alpha x}$ (واپاشی نمایی از یک هسته همدوس) است.

### ۶.۴ ارتباط با میدان لامبدا

UCC معادله پیوستگی برای جریان میدان لامبدا است ([[FRC-100-007]]). با استفاده از $\Lambda = \Lambda_0 \ln C$:

```
partial_t Lambda = -Lambda_0 div(J_C) + Lambda_0 S_C
```

جریان همدوسی می‌شود:

```
J_C^mu = -D_C partial^mu Lambda / Lambda_0
```

این شکل همورد UCC است که در تنظیمات نسبیتی معتبر است.

## ۷. اتلاف و پایداری

### ۷.۱ تابعی اتلاف

نرخ اتلاف کل را تعریف کنید:

```
sigma(t) = k* D_C integral(||grad(ln C)||^2 dV) >= 0
```

این تابعی همیشه غیرمنفی است و تضمین می‌کند که دینامیک UCC هرگز قانون دوم را نقض نمی‌کند. نرخ اتلاف اندازه می‌گیرد که گرادیان‌های همدوسی چقدر سریع هموار می‌شوند.

### ۷.۲ تابعی لیاپانوف

تابعی انرژی آزاد:

```
F[C] = -k*T integral(ln C dV)
```

یک تابعی لیاپانوف برای دینامیک UCC است:

```
dF/dt = -sigma(t) <= 0
```

این ثابت می‌کند که دینامیک UCC از نظر ترمودینامیکی پایدار است: سیستم همیشه به سمت انرژی آزاد پایین‌تر (همدوسی بالاتر) تکامل می‌یابد تا زمانی که اتلاف در تعادل ناپدید شود.

### ۷.۳ شرایط تعادل

در تعادل ($\sigma = 0$):

```
grad(ln C) = 0  =>  C = const. (همدوسی یکنواخت)
```

یا در حضور چشمه‌ها:

```
D_C nabla^2(ln C) = -S_C    (معادله پواسون برای همدوسی)
```

پروفایل همدوسی تعادلی توسط توزیع فضایی چشمه‌ها و چاه‌ها تعیین می‌شود.

## ۸. ساختار چند مقیاسی

### ۸.۱ سلسله مراتب سطح مو

UCC در مقیاس‌های متعدد عمل می‌کند که توسط **سطوح مو** در چارچوب FRC سازماندهی شده‌اند:

| سطح مو | مقیاس | $D_C$ | سیستم فیزیکی |
|:---|:---|:---|:---|
| $\mu_0$ | پلانک | $\sim \hbar/m_P$ | خلاء کوانتومی |
| $\mu_1$ | اتمی | $\sim 10^{-10}$ m²/s | الکترون‌ها، فوتون‌ها |
| $\mu_2$ | مولکولی | $\sim 10^{-6}$ m²/s | زیست‌مولکول‌ها |
| $\mu_3$ | عصبی | $\sim 10^{-2}$ m²/s | دینامیک مغز |
| $\mu_4$ | اجتماعی | $\sim 10^2$ m²/s | رفتار جمعی |

ضریب نفوذ $D_C$ با مقیاس افزایش می‌یابد که نشان‌دهنده انتقال سریع‌تر همدوسی در مقیاس‌های بزرگتر است.

### ۸.۲ جفت‌شدگی مقیاس

سطوح مو مجاور از طریق جمله چشمه جفت می‌شوند:

```
S_C^(n) = eta_{n,n-1} (C^(n-1) - C^(n)) + eta_{n,n+1} (C^(n+1) - C^(n))
```

این یک آبشار ایجاد می‌کند: همدوسی در مقیاس‌های ریز ($\mu_0$) چشمه همدوسی در مقیاس‌های درشت‌تر ($\mu_1, \mu_2, ...$) است، در حالی که واهمدوسی در مقیاس‌های درشت ($\mu_3$) همدوسی را در مقیاس‌های ریزتر از بین می‌برد.

## ۹. کاربردها

### ۹.۱ اندازه‌گیری کوانتومی

در طول اندازه‌گیری ([[FRC-100-003]]):
- همدوسی محلی به سرعت افزایش می‌یابد ($C_{sys} \to 1$ برای حالت ویژه انتخاب شده)
- همدوسی محیطی کاهش می‌یابد ($C_{env} \to 0$)
- کل: $dS_{total} + k^* d \ln C_{total} \geq 0$ (برآورده شده)

### ۹.۲ سیستم‌های زیستی

سیستم‌های زنده $C > 0$ را در برابر واهمدوسی حرارتی با تولید مداوم آنتروپی حفظ می‌کنند:

```
dS_env/dt >= k* |d(ln C_bio)/dt|
```

این یک **حداقل نرخ متابولیک** برای حفظ همدوسی تعیین می‌کند ([[FRC-100-005]]). 

### ۹.۳ رایانش کوانتومی

تصحیح خطای کوانتومی را می‌توان به عنوان حفظ همدوسی در برابر اتلاف UCC دید:

```
P_error_correction >= sigma(t) / k*T
```

توان مورد نیاز برای تصحیح خطا باید از نرخ اتلاف همدوسی فراتر رود.

## ۱۰. پیش‌بینی‌های تجربی

### ۱۰.۱ کمیت‌های قابل اندازه‌گیری

UCC پیش‌بینی‌هایی برای موارد زیر دارد:

1. **نرخ‌های واهمدوسی:** $\gamma_{decoherence} = D_C k_{min}^2$ (مرتبط با حداقل عدد موج سیستم)
2. **پروفایل‌های همدوسی:** واپاشی نمایی $C(r) \propto e^{-r/\xi_C}$ با طول همدوسی $\xi_C = \sqrt{D_C / \gamma}$
3. **تغییرات انرژی آزاد:** $\Delta G = -k^*T \Delta \ln C$ (قابل اندازه‌گیری با کالری‌سنجی)
4. **انحرافات قاعده بورن:** همدوسی خارج از تعادل $\delta P \in [10^{-4}, 10^{-3}]$ تولید می‌کند ([[FRC-100-006]])

### ۱۰.۲ تمایز FRC از واهمدوسی استاندارد

| مشاهده‌پذیر | واهمدوسی استاندارد | پیش‌بینی UCC |
|:---|:---|:---|
| پروفایل واپاشی | نمایی در زمان | نمایی در فضا و زمان |
| وابستگی به چشمه | فقط محیط | خود-چشمه (جمله گرادیان) |
| تعادل | $C = 0$ (حرارتی) | $C > 0$ (مینیمم میدان لامبدا) |
| بازهمدوسی | ممنوع | مجاز (پژواک‌های همدوسی) |

متمایزترین پیش‌بینی **پژواک‌های همدوسی** است: بازیابی جزئی همدوسی پس از اندازه‌گیری تحت پروتکل‌های خاص معکوس کردن فاز.

## ۱۱. نتیجه‌گیری

تقابل آنتروپی-همدوسی ($dS + k^* d \ln C = 0$) و شرایط همدوسی جهانی ستون فقرات ترمودینامیکی چارچوب FRC را فراهم می‌کنند. با هم آنها تضمین می‌کنند:

1. **سازگاری ترمودینامیکی:** بدون حرکت دائمی، بدون کاهش آنتروپی در انزوا
2. **محتوای دینامیکی:** یک PDE حاکم بر جریان همدوسی در فضا و زمان
3. **قدرت پیش‌بینی:** پیش‌بینی‌های خاص و قابل اندازه‌گیری برای نرخ‌های واهمدوسی، پروفایل‌های همدوسی و انحرافات قاعده بورن
4. **کاربرد جهانی:** معادلات یکسان از مقیاس‌های کوانتومی تا ماکروسکوپیک عمل می‌کنند

UCC معادله اصلی همدوسی است. تمام نتایج دیگر FRC—فروپاشی ([[FRC-100-003]])، قاعده بورن ([[FRC-100-006]])، دینامیک میدان لامبدا ([[FRC-100-007]])—موارد خاص یا پیامدهای این معادله جریان بنیادی هستند.
