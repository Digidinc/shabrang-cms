# FRC 100.005 — سازگاری ترمودینامیکی میدان همدوسی

## ۱. مقدمه

هر اصلاحی در دینامیک کوانتومی باید از نظر ترمودینامیکی سازگار باشد—نباید اجازه ساخت ماشین‌های حرکت دائمی را بدهد یا قانون دوم را نقض کند. از آنجایی که FRC یک متغیر دینامیکی جدید (میدان لامبدا) و یک نیروی محرکه جدید (گرادیان همدوسی) را معرفی می‌کند، باید نشان دهیم که این اضافات پارادوکس‌های ترمودینامیکی ایجاد نمی‌کنند.

این مقاله ثابت می‌کند که دینامیک FRC قانون دوم را دقیقاً برآورده می‌کند. "هزینه" حفظ یا افزایش همدوسی ($d \ln C > 0$) با افزایش متناظر در آنتروپی محیط ($dS_{env}$) پرداخت می‌شود، به طوری که آنتروپی کل هرگز کاهش نمی‌یابد.

![سازگاری ترمودینامیکی](/images/slides_raw/The_Illusion_of_Chance/page_3_img_0_Im1.png)

## ۲. جفت‌شدگی آنتروپی-همدوسی

### ۲.۱ آنتروپی تعمیم‌یافته

ما آنتروپی تعمیم‌یافته کل را به صورت زیر تعریف می‌کنیم:

```
S_total = S_sys + S_env - k* ln C
```

که در آن $k^*$ ثابت همدوسی است (در زمینه‌های ترمودینامیکی برابر با $k_B$). جمله $-k^* \ln C$ نشان‌دهنده "بدهی آنتروپی" پرداخت شده برای حفظ همدوسی است.

### ۲.۲ قانون تقابل

از اصول اولیه FRC ([[FRC-566-001]])، ما استخراج می‌کنیم:

```
dS + k* d ln C = 0    (سیستم بسته، برگشت‌پذیر)
```

این تقابل آنتروپی-همدوسی است: هر افزایش در همدوسی مستلزم کاهش جبرانی در آنتروپی است (یا بالعکس). برای فرآیندهای برگشت‌ناپذیر:

```
dS_total = dS_sys + dS_env + k* d ln C >= 0
```

نابرابری تضمین می‌کند که قانون دوم همیشه برآورده می‌شود.

### ۲.۳ تفسیر فیزیکی

| فرآیند | $dS$ | $d \ln C$ | اثر خالص |
|:---|:---|:---|:---|
| اندازه‌گیری (فروپاشی) | افزایش می‌یابد | به طور محلی افزایش می‌یابد | آنتروپی به محیط صادر می‌شود |
| واهمدوسی | افزایش می‌یابد | کاهش می‌یابد | فرآیند استاندارد قانون دوم |
| قفل فاز | به طور محلی کاهش می‌یابد | افزایش می‌یابد | متعادل شده توسط $dS_{env}$ محیطی |
| تعادل حرارتی | ماکزیمم | مینیمم ($C \to 0$) | حالت پایه تقابل |

## ۳. انرژی آزاد تعمیم‌یافته

### ۳.۱ استخراج

برای یک فرآیند همدما، انرژی آزاد هلمهولتز به صورت زیر تعمیم می‌یابد:

```
F = U - TS + k*T ln C
```

شرط تعادل به این صورت می‌شود:

```
dF = 0  =>  dU = T dS - k*T d ln C
```

این یک پتانسیل ترمودینامیکی جدید را تعریف می‌کند که همدوسی را در نظر می‌گیرد.

### ۳.۲ رابطه گیبس-همدوسی

در فشار و دمای ثابت، انرژی آزاد گیبس به صورت زیر می‌شود:

```
G = H - TS - k*T ln C
```

شرط خودبه‌خودی بودن عبارت است از:

```
deltaG = -k*T delta(ln C) < 0
```

فرآیندهایی که همدوسی را افزایش می‌دهند از نظر ترمودینامیکی مطلوب هستند—آنها انرژی آزاد تعمیم‌یافته را کاهش می‌دهند. این یک "پیکان" ترمودینامیکی را به سمت همدوسی نشان می‌دهد.

![چشم‌انداز انرژی آزاد](/images/slides_raw/The_Illusion_of_Chance/page_5_img_0_Im1.png)

## ۴. هزینه همدوسی

### ۴.۱ بودجه انرژی

حفظ همدوسی نیاز به انرژی دارد. توان تلف شده برای حفظ سطح همدوسی $C$ در برابر واهمدوسی محیطی عبارت است از:

```
P_coherence = k*T * gamma_decoherence * |ln C|
```

که در آن $\gamma_{decoherence}$ نرخ واهمدوسی است. برای سیستم‌های کوانتومی در دمای اتاق، این هزینه ناچیز است ($\sim 10^{-20}$ W)، اما برای همدوسی ماکروسکوپیک قابل توجه می‌شود.

### ۴.۲ حد ترمودینامیکی بر همدوسی

حداکثر همدوسی قابل دستیابی در دمای $T$ محدود است به:

```
C_max = exp(-Delta E / (k*T))
```

که در آن $\Delta E$ شکاف انرژی بین حالت همدوس و حالت پایه حرارتی است. این همدوسی را به توزیع بولتزمن متصل می‌کند و توضیح می‌دهد که چرا همدوسی کوانتومی ماکروسکوپیک در دماهای بالا نادر است.

### ۴.۳ همدوسی و پیکان زمان

تقابل آنتروپی-همدوسی دیدگاه جدیدی در مورد پیکان زمان ارائه می‌دهد:

- در **جهت ترمودینامیکی:** آنتروپی افزایش می‌یابد، همدوسی کاهش می‌یابد
- در **جهت بیولوژیکی:** همدوسی به قیمت تولید آنتروپی محیطی حفظ می‌شود
- در **جهت کیهان‌شناسی:** حالت اولیه با آنتروپی کم با یک شرط اولیه با همدوسی بالا مطابقت دارد

## ۵. قضایای نوسان

### ۵.۱ قضیه کروکس-همدوسی

قضیه نوسان کروکس تعمیم می‌یابد تا شامل همدوسی شود:

```
P(deltaS + k* delta ln C = A) / P(deltaS + k* delta ln C = -A) = exp(A / k*T)
```

این تضمین می‌کند که حتی در سطح میکروسکوپی، تقابل آنتروپی-همدوسی از نظر آماری برقرار است.

### ۵.۲ برابری جارزینسکی-همدوسی

برابری جارزینسکی به صورت زیر می‌شود:

```
<exp(-(W - deltaF) / (k*T))> = <exp(k* delta ln C / (k*T))>
```

که اندازه‌گیری‌های کار غیرتعادلی را به تغییرات همدوسی متصل می‌کند.

## ۶. پیامدها

### ۶.۱ موتورهای حرارتی کوانتومی

موتورهای حرارتی کوانتومی سازگار با FRC می‌توانند کار را از همدوسی استخراج کنند:

```
W_extracted = k*T delta ln C    (حد ماکزیمم، برگشت‌پذیر)
```

این منبع جدیدی برای پروتکل‌های ترمودینامیک کوانتومی فراهم می‌کند.

### ۶.۲ سیستم‌های زیستی

سیستم‌های زنده با تولید مداوم آنتروپی، همدوسی ($C > 0$) را دور از تعادل حرارتی حفظ می‌کنند. چارچوب FRC این را کمی می‌کند:

```
dS_env/dt >= k* |d ln C_bio/dt|
```

این حداقل نرخ متابولیک را برای حفظ همدوسی بیولوژیکی تعیین می‌کند.

## ۷. اثبات انطباق با قانون دوم

### ۷.۱ بیان رسمی

**قضیه:** تحت دینامیک FRC، آنتروپی تعمیم‌یافته کل $S_{total} = S_{sys} + S_{env} - k^* \ln C$ برای هر فرآیندی غیرکاهشی است.

**طرح اثبات:**

1. دینامیک میدان لامبدا $S_{total}$ را برای فرآیندهای برگشت‌پذیر حفظ می‌کند (توسط تقابل)
2. فرآیندهای برگشت‌ناپذیر $\Delta S_{irr} \geq 0$ را اضافه می‌کنند (از ترمودینامیک استاندارد)
3. اختلال FRC به همیلتونی هرمیتی است (انرژی را حفظ می‌کند)
4. بنابراین $\Delta S_{total} = \Delta S_{irr} \geq 0$ برای همه فرآیندها

### ۷.۲ بدون حرکت دائمی

FRC امکان‌پذیر نمی‌کند:
- **نوع ۱:** انرژی حفظ می‌شود (دینامیک هرمیتی میدان لامبدا)
- **نوع ۲:** تقابل آنتروپی-همدوسی از استخراج همدوسی بدون هزینه آنتروپی جلوگیری می‌کند
- **نوع ۳:** اصطکاک و واهمدوسی حداقل کار برگشت‌ناپذیر را تعیین می‌کنند

## ۸. نتیجه‌گیری

چارچوب FRC کاملاً با قوانین ترمودینامیک سازگار است. میدان لامبدا و دینامیک همدوسی یک متغیر ترمودینامیکی جدید ($C$) با هزینه آنتروپی کاملاً تعریف شده ($-k^* \ln C$)، یک انرژی آزاد تعمیم‌یافته ($G = H - TS - k^*T \ln C$) و قضایای نوسان را معرفی می‌کنند که وقتی $C = 1$ به نتایج استاندارد کاهش می‌یابند. هیچ ماشین حرکت دائمی در FRC ممکن نیست.
