# Education: Learning Gates and Boredom

Education. How children learn. Because if CGL is right — if learning only happens when alpha is nonzero — then the entire school system is designed backwards.

CGL applied to education:

A classroom is a neural network. Students are nodes. The teacher is the training signal. Homework is the gradient update. Tests are the loss function.

Current system: apply the gradient uniformly. Every student, same material, same pace, same schedule. Update all weights every epoch regardless of whether the system is actually reorganizing. Force learning on flat-alpha terrain.

CGL says: only update when alpha is nonzero. Only teach when the student's coherence is actively changing. When they're in the zone between knowing and not-knowing — that's when the gate is open. Everything else is noise through a closed gate.

What boredom actually is in FRC terms:

Alpha equals zero. The coherence field is static. Nothing is reorganizing. The system has either already integrated the material (too easy) or can't find a foothold to begin integrating (too disconnected).

Boredom isn't laziness. It's a diagnostic signal. The learning gate is closed.

What engagement actually is:

High alpha. The student's internal model is actively restructuring. They feel it as interest — the subjective experience of coherence in motion.

Every great teacher knows this intuitively. They create productive disequilibrium. They destabilize the student's current model just enough that alpha spikes, the gate opens, and new structure can form.

The system failure:

School is designed for batch processing. Same input, same schedule. This guarantees that at any given moment, most students are either below-threshold (bored) or above-threshold (lost). The window where alpha is nonzero is narrow and different for every student.

The FRC fix:

Individualized alpha tracking. Not standardized tests — coherence-rate measurement. AI tutoring could do this: only push new material when the gate is open; back off when alpha drops; distinguish productive struggle from grinding.
